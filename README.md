# Home Loan Calculator

A simple Python program that calculates the interest and total repayment amount for a personal loan.  
The user can choose between simple interest or compound interest, and the program outputs both the interest and the total amount to be paid.  

#How to use the code 

Run the program in a Python environment and enter the interest type when prompted:
- simple → calculates using the simple interest formula  
- compound → calculates using the compound interest formula
